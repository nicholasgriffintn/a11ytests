Document Language Issues

WCAG 2.1 Success Criterion 3.1.1 Language of Page (Level A): The default human language of each page can be programmatically determined.

WCAG 2.1 Success Criterion 3.1.2 Language of Parts (Level AA): The human language of each passage or phrase in the content can be programmatically determined.

This page demonstrates issues with document language specification.

Missing HTML Lang Attribute

This page has a missing lang attribute in the HTML tag. Screen readers will default to the user's system language, potentially causing incorrect pronunciation.

Mixed Language Content Without Lang Attributes

This paragraph contains French: Bonjour, comment allez-vous? But it doesn't use the lang attribute to specify it.

This paragraph contains Spanish: Hola, ¿cómo estás? But it doesn't use the lang attribute to specify it.

Incorrect Language Code

This paragraph uses an invalid language code "xx" which doesn't conform to BCP 47 standards.

Multiple Languages on One Page

When a page contains content in multiple languages, each section in a different language should use the lang attribute:

Ich spreche ein bisschen Deutsch.

Parlo un po' di italiano.
